ชื่อโปรเจค : Useless Box

โปรเจควิชา : DMT350(Media Hardware Technology)

รูปแบบโปรเจค : โปรเจคเกี่ยวกับการออมเงิน

รายละเอียดโปรเจค : Useless Box เป็นโปรเจคสำหรับการฝึกให้เด็กออมเงิน โดยตัวโปรเจค Useless Box นี้ ใช้ Arduino ควบคุมการทำงานของโปรเจค

เช่น การเปิด-ปิดฝากล่องและแขนของตัวตุ๊กตาก็จะใช้อุปกรณืที่ชื่อว่า "Survo" พร้อมเขียนCode ในการควบคุมการทำงานของกล่องUseless Box
